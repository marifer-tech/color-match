Color Match Reflex
Mini game de atenção visual e controle cognitivo
Efeito Stroop
Clique na cor certa, não na cor que você vê.

A palavra mostra uma cor, mas está pintada com outra. Seu desafio é escolher o botão da cor escrita, ignorando o estímulo visual.

Como funciona?

1. Uma palavra de cor aparece no centro da tela.
2. Você deve clicar no botão cuja cor escrita corresponde à palavra.
3. O tempo vai diminuindo e a dificuldade aumenta.

Exemplo: a palavra AZUL aparece em vermelho. A resposta correta é o botão AZUL.
Fluxo para prints: Tela inicial → Tela de jogo → Tela de resultado.
